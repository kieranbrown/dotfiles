# Dotfiles

## Install

```shell
DOTFILES_DIR="${HOME}/Code/github.com/kieranbrown/dotfiles"
mkdir -p "${DOTFILES_DIR}"
git clone https://github.com/kieranbrown/dotfiles.git "${DOTFILES_DIR}"
cd "${DOTFILES_DIR}"
./etc/install.sh
```
